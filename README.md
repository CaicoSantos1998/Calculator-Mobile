# Calculator-Mobile
This is a basic Java calculator.

## About Calculator
The calculator is for the Android operating system and currently performs the four basic operations.

## Technologies
* **Language:** Java 21.
* **Android:** Android 7.0.

## Setup
1 - Clone the repository.

2 - You can pair it with your smartphone to start the project or use your PC.
